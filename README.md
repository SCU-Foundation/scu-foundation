# SCU Foundation

The SCU Foundation is the neutral, non‑commercial governance root of the Lattice Protocol.  
We steward open standards, certification, institutional governance, and long‑term public‑good infrastructure for the AI‑native internet.

## Governance Documents

### Conceptual Frame  
Defines the purpose, category, and governance role of the Lattice Protocol.  
https://latticeprotocol.org/governance/conceptual-frame/

### Architecture Model  
Outlines the seven‑layer structure of the AI‑native internet, from governance root to application layer.  
https://latticeprotocol.org/governance/architecture-model/

## Scope

- Governance and standards stewardship  
- Certification and compliance  
- Long‑term public‑good infrastructure  
- Preservation of the SCU architectural definition
