# SCU Foundation Website

The SCU Foundation maintains the canonical Superconducting Compute Unit (SCU) architecture and its associated publications.  
This repository contains the source files for the official website at **https://scu.foundation**.

## Purpose
- Provide stable, citable access to the SCU Whitepaper  
- Maintain version history and archival materials  
- Host supporting publications, diagrams, and technical notes  
- Present institutional information about the SCU Foundation

## Development
This site is built as a static HTML/CSS project with no client‑side frameworks.  
All pages are hand‑authored for clarity, longevity, and institutional stability.
